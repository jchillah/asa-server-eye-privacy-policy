Datenschutzerklärung für ASA Server Eye

Wichtiger Hinweis: Diese Datenschutzerklärung gilt für die mobile App „ASA Server Eye“ für Android und beschreibt, welche Daten verarbeitet werden und zu welchen Zwecken dies erfolgt.

Die App ASA Server Eye wurde von Michael Winkler entwickelt. Der Schutz Ihrer personenbezogenen Daten ist uns wichtig. Diese Datenschutzerklärung informiert Sie über Art, Umfang und Zweck der Verarbeitung personenbezogener Daten im Zusammenhang mit der Nutzung der App.

3. Welche Daten verarbeitet werden

3.1 Technische und automatisch übermittelte Daten

  • Geräteinformationen, z. B. Gerätemodell und Betriebssystem-Version
  • App-Version und technische Diagnoseinformationen
  • IP-Adresse oder vergleichbare netzwerkbezogene Informationen
  • Nutzungsdaten zur Bereitstellung und Verbesserung der App

3.2 Daten innerhalb der App

  • Von Ihnen gespeicherte Favoriten
  • App-Einstellungen und Präferenzen
  • Anmeldeinformationen, sofern Login-Funktionen verwendet werden

3.3 Werbe- und Analyseinformationen

  • Werbe-ID bzw. Advertising ID, soweit vom Gerät bzw. System bereitgestellt
  • Interaktionen mit Werbeanzeigen
  • Nutzungs- und Diagnoseinformationen zur App-Verbesserung

4. Zwecke der Datenverarbeitung

Die Verarbeitung der Daten erfolgt insbesondere zu folgenden Zwecken:

  • Bereitstellung der Funktionen von ASA Server Eye
  • Anzeige und Aktualisierung von Serverinformationen
  • Speicherung von Favoriten und Einstellungen
  • Verbesserung von Stabilität, Sicherheit und Performance der App
  • Einblendung von Werbung zur Finanzierung der App

5. Eingesetzte Dienste

5.1 Firebase

Diese App verwendet Dienste von Firebase, einem Angebot von Google. Je nach eingebundener Funktion können dabei insbesondere Nutzungsdaten, Geräteinformationen und technische Fehlerberichte verarbeitet werden.

Weitere Informationen finden Sie unter: firebase.google.com/support/privacy

5.2 Google AdMob

Diese App verwendet Google AdMob, um Werbeanzeigen in der App anzuzeigen. Dabei können insbesondere Gerätekennungen, Werbe-IDs, Nutzungsdaten und Interaktionen mit Werbung verarbeitet werden.

Weitere Informationen finden Sie unter: policies.google.com/privacy

5.3 Firebase Authentication

Sofern in der App eine Anmeldung angeboten wird, können im Rahmen der Authentifizierung entsprechende Anmeldedaten verarbeitet werden, um Benutzerkonten bereitzustellen und zu verwalten.

6. Weitergabe von Daten

Es erfolgt kein Verkauf Ihrer personenbezogenen Daten. Eine Weitergabe kann nur erfolgen, soweit dies technisch erforderlich ist, insbesondere an folgende Dienstleister:

  • Google / Firebase
  • Google AdMob

7. Speicherung und Löschung

  • Lokal gespeicherte Daten verbleiben grundsätzlich auf Ihrem Gerät, bis sie gelöscht werden.
  • Beim Deinstallieren der App können lokal gespeicherte App-Daten entfernt werden.
  • Daten bei Drittanbietern werden nach deren jeweiligen Datenschutz- und Aufbewahrungsrichtlinien verarbeitet.
